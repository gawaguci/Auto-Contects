## 1. Visual Theme & Atmosphere

Auto Content는 유튜브 영상 자동 생성 파이프라인의 웹 UI다. 기존 CLI 환경을 대체하는 이 인터페이스는 콘텐츠 제작자가 빠르고 직관적으로 영상을 설정하고 생성 과정을 모니터링할 수 있도록 설계됐다. 디자인 언어는 차분하고 집중력을 유지시키는 방향으로, 눈의 피로를 최소화하는 소프트 블루/틸 계열(`oklch(0.6 0.1 250)`)을 기반으로 한다.

## 7. Do's and Don'ts

### Do
- `box-shadow: 0 0 0 1px` 방식으로 인터랙티브 요소 테두리 처리 (상태 변경 시 레이아웃 이동 없음)
- 컴포넌트에서는 시맨틱 토큰만 사용 (`--text-primary`, not `--color-neutral-900`)
- CSS `:hover`, `:active` 사용 — JS 이벤트 핸들러 아님
- Named React imports: `import { forwardRef, useId } from "react"`
- `oklch()`로 모든 색상 값 표현 — 토큰 정의에 hex/hsl 금지
- APCA Lc ≥ 60 대비 검증

### Don't
- `React.FC`, `React.ReactNode`, `React.ElementRef` 사용 금지 (named imports 사용)
- 컴포넌트에 색상 하드코딩 금지 (항상 시맨틱 토큰 사용)
- Input/Select에 CSS `border` 사용 금지 (box-shadow 사용)
- 인터랙티브 상태에 JS 상태 관리 사용 금지 (CSS :hover/:active 사용)
- 컴포넌트에서 프리미티브 토큰 직접 사용 금지 (`--color-primary-600` → `--interactive-primary`)

## 8. Responsive Behavior

### Breakpoints
| Name | Width | Key Changes |
|------|-------|-------------|
| Mobile | < 640px | 단일 컬럼, 스택 레이아웃 |
| Tablet | 640–1024px | 2컬럼, 축약 네비게이션 |
| Desktop | > 1024px | 전체 레이아웃, 확장 네비게이션 |

### Touch Targets
- 최소 탭 타겟: `44×44px`
- 모든 크기에서 버튼 최소 높이 보장

### Collapsing Strategy
- 네비게이션: 가로형 → 모바일에서 햄버거
- 카드 그리드: 다중 컬럼 → 단일 컬럼
- 다이얼로그: `w-full max-w-[Xpx]` with 모바일 수평 마진
